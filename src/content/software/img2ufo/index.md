img2ufo is a Rust tool that converts raster images directly into UFO (Unified Font Object) source files. It streamlines the workflow of going from hand-drawn or AI-generated letterforms to editable font sources.

- **Language:** Rust
- **Source:** [github.com/eliheuer/img2ufo](https://github.com/eliheuer/img2ufo)
