img2bez is a Rust library and command-line tool for tracing raster images into font-ready bezier outlines. It's designed for type designers who want to convert hand-drawn letterforms or other raster artwork into clean vector paths suitable for font production.

- **Language:** Rust
- **Source:** [github.com/eliheuer/img2bez](https://github.com/eliheuer/img2bez)
