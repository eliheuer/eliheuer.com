Bez is an experimental font editor built with GPUI, the UI framework from the Zed code editor. It explores using Zed's high-performance GPU-accelerated rendering for font editing.

- **Language:** Rust
- **UI Framework:** GPUI (Zed)
- **Source:** [github.com/eliheuer/bez](https://github.com/eliheuer/bez)
