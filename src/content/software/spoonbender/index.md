Spoonbender is a cross-platform UFO font editor built in Rust, forked from the original Runebender project. It uses the Druid UI framework and reads/writes the UFO font format.

- **Language:** Rust
- **UI Framework:** Druid
- **Source:** [github.com/eliheuer/spoonbender](https://github.com/eliheuer/spoonbender)
