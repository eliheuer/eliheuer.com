Bezy is an experimental font editor built with the Bevy game engine. It explores using a game engine's rendering pipeline and ECS architecture for interactive font editing.

- **Language:** Rust
- **UI Framework:** Bevy
- **Source:** [github.com/eliheuer/bezy](https://github.com/eliheuer/bezy)
