Micro Grotesk is an experimental typeface designed at 256 units per em, targeting the smallest possible file size. It explores the limits of how small a functional font file can be while remaining legible.

- **Source:** [github.com/eliheuer/micro-grotesk](https://github.com/eliheuer/micro-grotesk)
